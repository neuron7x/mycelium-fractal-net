
# VALIDATION_NOTES — MyceliumFractalNet v4.1

Цей документ описує очікувані діапазони метрик, а не зафіксовані "магічні" числа.
Мета — мати фізично і інженерно правдоподібні результати, стійкі по seed.

> **Для повної математичної формалізації** з явними рівняннями, таблицями параметрів,
> одиницями вимірювання та допустимими діапазонами див. **[MFN_MATH_MODEL.md](MFN_MATH_MODEL.md)**.
>
> Для архітектурних деталей див. **[ARCHITECTURE.md](ARCHITECTURE.md)**.

## 1. Нернст

- Іон: K⁺
- [K⁺]_in = 140 mM
- [K⁺]_out = 5 mM
- T = 310 K

Очікуємо:
- E_K ≈ -89 мВ (допуск ±5 мВ).

## 2. Дифузійна решітка

Для `grid_size=64`, `steps=64`:

- потенціали в діапазоні приблизно [-95, 40] мВ;
- розподіл без NaN/Inf;
- хоча б декілька growth_events за симуляцію (залежить від `spike_probability`).

## 3. Фрактальна розмірність

Для типових полів при порозі -60 мВ:

- D у межах 1.4–1.9;
- невелика варіативність між seed (std ≈ 0.01–0.1).

## 4. Навчання MyceliumFractalNet

- `loss_start` > `loss_final`;
- `loss_final` має бути додатною, але менше початкової;
- при seed=42 зазвичай спостерігається drop на 20–40 %.

Ці діапазони можна оновлювати на основі емпіричних запусків і включати у
більш суворі property-based тести.
